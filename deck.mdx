# Gatbsy

GatsbyJS, est un générateur de site statique construit avec ReactJS et alimenté par GraphQL. Gatsby permet à quiconque de créer des sites Web et des applications riches en fonctionnalités et attrayantes.

## Les nouveaux modes de rendu

<ul>
  <li>SSG - static site generation</li>
  <li>DSG - deferred static generation</li>
  <li>SSR - server-side rendered</li>
</ul>

---
